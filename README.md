# 动态表单引擎

可配置化的动态表单引擎，基于Vue3、Vite、TypeScript、Nodejs、Koa、MongoDB开发。
[https://tuziki.com/demo/formEngine/](https://tuziki.com/demo/formEngine/)


## 功能特性

### 表单设计

- 拖拽式表单设计
- 支持表单布局
- 支持表单字段类型
- 支持表单字段校验
- 支持表单字段联动
- 支持表单字段属性
- 支持表单字段事件
- 支持表单字段样式
- 支持表单字段注释
- 支持表单字段分组
- 支持表单字段排序
- 支持表单字段配置
- 支持表单字段扩展

### 数据渲染
- 支持低代码配置
- 支持无代码配置
